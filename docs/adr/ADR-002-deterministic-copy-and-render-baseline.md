# ADR-002 Phase 1 카피/렌더 안정성 기준선

- 상태: 승인
- 날짜: 2026-04-07

## 배경

기획 문서는 `Gemma 4`를 포함한 모델 후보군을 유지하되, 발표 안정성을 최우선으로 두고 비교 후 기본값을 정하도록 요구합니다.  
현재 저장소는 모델 어댑터와 벤치마크 기반이 없고, Phase 1 Must에는 안정적인 숏폼/게시글 생성과 업로드 흐름 시연이 우선입니다.

## 결정

1. Phase 1 기본 카피 생성은 `packages/template-spec` 기반의 규칙형 생성기로 시작합니다.
2. 숏폼 렌더는 FFmpeg/Pillow 기반의 deterministic scene renderer를 기본 경로로 사용합니다.
3. 모델 기반 카피 생성과 고급 멀티모달 판단은 후속 실험 항목으로 분리합니다.
4. 업로드는 실제 외부 채널 API 대신 `instagram` 채널에 대해 시뮬레이션 가능한 adapter와 `assist fallback`을 기본 제공 경로로 둡니다.

## 근거

1. 같은 입력에서 같은 결과를 재현하기 쉽습니다.
2. 실패 원인을 step 단위로 분리하기 쉽습니다.
3. 템플릿/스타일/quick action 정책을 먼저 검증할 수 있습니다.

## 결과

1. 모델 품질 향상 여지는 남지만, 현재 단계의 데모 안정성은 높아집니다.
2. 실제 `instagram` API 연동과 `Gemma 4` 벤치마크는 별도 실험/확장 작업으로 남습니다.
3. 향후 기본 생성 모델을 채택할 때는 `docs/experiments/` 비교 기록과 새 ADR이 필요합니다.
