# archive 가이드

`docs/archive/`는 현재 trunk 설명의 본문이 아니라, 루트에서 내려 보낸 보관 자료를 두는 구역입니다.

## 들어 있는 문서

- [HISTORY.md](HISTORY.md)  
  세션 누적 이력 요약입니다.
- [planning-brief-initial.md](planning-brief-initial.md)  
  초기 서비스 기획 초안입니다.

## 왜 따로 두었는가

- 포트폴리오 첫인상에서는 현재 trunk와 직접 연결되는 문서가 먼저 보여야 합니다.
- 하지만 초기 기획과 누적 이력도 나중에 맥락을 복원할 때는 필요합니다.

그래서 이 문서들은 삭제하지 않고, 메인 동선 밖으로 내려 보관합니다.
