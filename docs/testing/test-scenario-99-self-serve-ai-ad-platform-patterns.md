# Test Scenario 99 - self-serve AI ad platform patterns

## 목적

- 외부 AI ad platform 분석이 현재 프로젝트의 제품 목적과 실제로 연결되는지 확인합니다.

## 입력 자료

1. `docs/experiments/EXP-96-reference-template-fit-scorecard.md`
2. `docs/experiments/EXP-97-self-serve-ai-ad-platform-patterns.md`
3. 외부 참고 링크
   - `https://shortvideo.ai/`
   - `https://www.shortbox.ai/`
   - `https://www.quickads.ai/video-ads`
   - `https://www.capcut.com/tools/ai-commercial`
   - `https://www.genre.ai/`

## 절차

1. `EXP-97`을 읽고 외부 제품들이 공통으로 무엇을 제품화하는지 확인합니다.
2. 아래 항목이 실제로 self-serve 제품 핵심으로 정리됐는지 체크합니다.
   - 입력 단순화
   - 슬롯화
   - 수정 가능한 패키지
   - 채널 최적화
   - variation 생성
3. 현재 프로젝트의 본선과 비교해 gap 항목이 구체적인지 확인합니다.
4. 다음 우선순위가 `영상 품질 1편`보다 `템플릿 구조/스토리보드/슬롯화` 쪽으로 이동했는지 점검합니다.

## 기대 결과

1. 외부 플랫폼 분석이 단순 벤치마킹이 아니라 제품 방향 판단 자료로 기능합니다.
2. 현재 프로젝트의 본선 우선순위가 `self-serve template product` 기준으로 다시 정렬됩니다.
3. 연구선의 generated clip도 `완성형 영상`보다 `템플릿 슬롯용 asset` 관점으로 재정의됩니다.

## 판정 기준

- 아래 3개가 모두 맞으면 통과입니다.
  1. 외부 제품의 공통 패턴이 4개 이상 뽑혀 있는가
  2. 현재 프로젝트와의 gap이 제품 관점으로 적혀 있는가
  3. 다음 우선순위가 self-serve 목적과 직접 연결되는가
