# EXP-227 B-grade Quality Gate And Video Baseline Scorecard

## 1. 기본 정보

- 실험 ID: `EXP-227`
- 작성일: `2026-04-13`
- 작성자: Codex
- 관련 기능: `영상 baseline 판정 기준 / 의도된 B급 vs 실패한 생성 구분 / 다음 비교 기준선 고정`

## 2. 왜 이 문서를 만들었는가

- 지금까지의 논의에서 핵심은 `최신 모델을 쓰느냐`가 아니라, `이 서비스의 B급 광고 문법을 안정적으로 만들 수 있느냐`로 정리됐습니다.
- 따라서 다음 영상 baseline 비교 전에, 무엇을 `좋은 B급`으로 보고 무엇을 `실패한 생성`으로 보는지 먼저 고정할 필요가 있었습니다.
- 이번 문서는 새 모델을 추가하는 문서가 아니라, 이후 실험이 같은 기준으로 판정되도록 만드는 gate 문서입니다.

## 3. 이 프로젝트에서 말하는 B급 감성

- 이 프로젝트의 B급 감성은 `AI 한계 때문에 어쩔 수 없이 어색한 결과`가 아닙니다.
- 의도는 아래와 같습니다.
  1. 저예산 전단지/짧은 숏츠 같은 과장된 템포
  2. 과한 카피, 빠른 컷, 즉시 보이는 CTA
  3. 일부 촌스러운 연출을 오히려 의도된 광고 톤으로 흡수
- 따라서 B급 감성은 주로 `편집 템포`, `오버레이`, `카피 톤`, `컷 구성`에서 만들어져야 합니다.
- 반대로 음식/음료 구조가 깨지는 것은 B급이 아니라 실패입니다.

## 4. 의도된 B급 vs 실패한 생성

### 4.1 의도된 B급

- 과장된 headline이나 후킹 문구
- 빠른 컷 편집
- 투박한 zoom/crop
- 사장님이 직접 만든 듯한 전단지형 오버레이
- 템플릿 기반으로 반복 가능한 톤

### 4.2 실패한 생성

- 원본 음식 모양이 달라짐
- 컵/병/접시 구조가 변형됨
- 주변 오브젝트가 새로 생김
- 메뉴판 QR, 배경 텍스트, 로고 같은 요소가 이상한 위치로 이동함
- 원본에는 없는 재료/소품이 추가됨
- 영상이 멋져 보여도 매장/메뉴 정체성이 깨짐

## 5. 이번부터 고정할 판정 기준 4개

### 5.1 원본 보존성

- 메뉴와 매장의 정체성이 유지되는가
- 사용자가 올린 사진의 핵심 요소가 다른 물체로 바뀌지 않는가

### 5.2 광고 체감 품질

- 첫 1초에 메뉴/혜택/무드 중 하나가 즉시 꽂히는가
- 짧은 영상으로도 광고처럼 느껴지는가
- 단순한 정지사진 나열이 아니라, 최소한의 편집 에너지나 motion 체감이 있는가

### 5.3 반복 안정성

- 같은 설정을 다시 돌렸을 때 결론이 크게 바뀌지 않는가
- 특정 샘플에서만 우연히 좋아 보인 결과가 아닌가

### 5.4 서비스 적합성

- 템플릿, 카피, CTA, 결과물 패키징과 결합 가능한가
- 자영업자 셀프서브 흐름에서 통제 가능한가

## 6. 즉시 탈락시키는 red flag

다음 중 하나라도 발생하면 `production 후보`로는 탈락입니다.

1. 원본 음식/음료 형상이 달라짐
2. 배경 QR, 간판, 글자가 다른 위치의 핵심 오브젝트처럼 튀어나옴
3. 원본에 없던 소품이 전면 등장함
4. 라벨/브랜드/병/잔 구조가 뒤틀림
5. CTA나 템플릿 연출을 얹기 어려울 정도로 결과가 불안정함

## 7. 모델 역할 정의

### 7.1 product control

- 현재 서비스 본선 기준선입니다.
- 의미:
  - `template motion + compositor`
  - deterministic packaging
  - CTA/카피/컷 구성을 통제 가능한 baseline

### 7.2 실사용 후보 모델

- local LTX, Sora 2처럼 `보존성 + 반복성 + 서비스 적합성`을 같이 평가할 대상입니다.

### 7.3 upper-bound reference

- manual Veo처럼 `품질 상한선 참고`용입니다.
- 품질 참고에는 유효하지만, 보존성이 깨지면 production 후보가 아닙니다.

## 8. 다음 baseline 비교 규칙

다음 비교는 아래 질문만 답해야 합니다.

1. `product control`보다 광고 체감 품질이 실제로 더 나은가
2. 그 향상이 원본 보존 손실 없이 가능한가
3. 반복해도 같은 결론이 유지되는가
4. 결과를 서비스 패키징에 넣을 수 있는가

## 9. scorecard

다음 비교 문서에는 최소 아래 표를 채워야 합니다.

## 10. 결론

- 이 프로젝트는 `최신 모델 추격전`이 아닙니다.
- 기준은 `의도된 B급 광고를 안정적으로 반복 생산할 수 있는가`입니다.
- 따라서 다음 영상 실험은 반드시 `product control`을 포함하고, `좋은 B급`과 `실패한 생성`을 분리해서 기록해야 합니다.
