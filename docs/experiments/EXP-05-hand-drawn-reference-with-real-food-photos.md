# EXP-05 손그림 레퍼런스 + 실제 음식 사진 실험

## 1. 기본 정보

- 실험 ID: `EXP-05`
- 작성일: `2026-04-08`
- 작성자: Codex
- 관련 기능: 손그림 스타일 영상 실험, 실제 음식 사진 기반 시나리오

## 2. 실험 목적

- 사용자가 제공한 `손그림샘플.png` 스타일이 실제 음식 사진에도 잘 붙는지 확인합니다.
- synthetic 샘플이 아니라 실제 음식 사진 파일명을 그대로 활용해 더 현실적인 시나리오를 만듭니다.
- 현재 유망 후보인 `owner_made_safe_zone`과 손그림 메뉴판형 스타일을 one-variable-at-a-time으로 비교합니다.

## 6. 실험 절차

1. `docs/sample`의 실제 음식 사진과 손그림 스타일 레퍼런스를 확인했습니다.
2. `video_harness.py`에 `EXP-05`와 `hand_drawn_menu_board` variant를 추가했습니다.
3. `타코야키 + 맥주` 실제 사진 조합으로 실험을 실행했습니다.
4. 첫 씬 이미지를 직접 비교했습니다.

## 8. 관찰 내용

### 확인된 것

- 손그림 스타일은 실제 음식 사진에도 충분히 붙습니다.
- 특히 `타코야키 + 맥주`처럼 명확한 조합에서는 메뉴판형/포스터형 무드가 잘 살아납니다.
- 파일명 기반 음식명(`타코야키`, `맥주`)을 그대로 쓰는 방식은 시나리오를 더 현실적으로 만드는 데 유효했습니다.

### 중요한 판단

- **손그림 스타일은 분명히 유효한 스타일 축입니다.**
- 다만 현재 결과물은 숏폼 첫 장면보다, `메뉴 보드`, `정사각형 포스터`, `게시글 이미지` 쪽에 더 잘 맞습니다.
- 즉 손그림 스타일은 버릴 게 아니라, 적용 표면을 구분해서 써야 합니다.

## 9. 실패/제약

1. 현재 hand-drawn variant는 설명형 요소가 많아 숏폼 첫 컷에서 훅이 약해질 수 있습니다.
2. 상품 사진이 원형 접시 안에 들어가면서 임팩트가 약간 줄었습니다.
3. motion을 붙이지 않은 상태라 숏폼에서의 실제 주목도는 아직 미확인입니다.

## 10. 결론

- 가설 충족 여부: **충족**
- 판단:
  - 손그림 스타일은 앞으로 계속 실험할 가치가 있습니다.
  - 다만 우선 적용 대상은 숏폼 메인 컷보다는 포스터/피드/메뉴 보드 쪽이 더 적합합니다.
  - 숏폼에 쓰려면 사진 비중을 더 키우고 손그림 요소는 프레임/스티커 수준으로 줄이는 보정이 필요합니다.
