구현된 기능 전체 정리
백엔드 (services/api)
기능	파일	내용
HttpOnly 쿠키 세션	routes.py	로그인/가입 시 session 쿠키 발급, Bearer 토큰 완전 제거
argon2id 패스워드 해싱	security.py	가입 시 argon2id, 로그인 시 PBKDF2 → argon2id 자동 rehash
PIPA 4개 필수 동의	consent.py	terms/privacy/age_14/overseas_transfer + 버전/IP/UA 저장
만 14세 미만 차단	runtime.py	birthDate 기준 계산, 403 반환
계정 잠금 (5회 실패)	runtime.py	5회 실패 시 15분 잠금
레이트 리밋	rate_limit.py	로그인/가입/비밀번호 재설정 IP+이메일 기반
AES-256-GCM 토큰 암호화	crypto.py	OAuth 토큰 암호화 저장, 랜덤 IV
CSRF method 기반	main.py	POST/PATCH/DELETE/PUT 전부 적용, /api/webhooks/* 예외
소프트 삭제 + 30일 유예	runtime.py	deleted_at + hard_delete_at
하드 삭제 배치	hard_delete.py	hard_delete_at 도래한 계정 완전 삭제
비밀번호 변경 세션 제어	runtime.py	변경 시 현재 세션 유지, 나머지 전체 revoke
감사 로그	audit.py	register/login/logout 이벤트 기록
프론트엔드 (apps/web)
기능	파일	내용
로그인 페이지	login/page.tsx	쿠키 기반 로그인, ?logged_out=true 메시지
회원가입 페이지	register/page.tsx	4개 동의 체크박스, 생년월일, 비밀번호 10자 이상
로그아웃 라우트	logout/page.tsx	즉시 로그아웃 후 /login?logged_out=true 이동
AuthGate 단일화	auth-gate.tsx	로그인만 노출, register form 제거
이용약관 페이지	terms/page.tsx	미인증 접근 가능
개인정보처리방침	privacy/page.tsx	미인증 접근 가능
